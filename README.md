# dotfiles-2018

My dotfiles
## Toolset
- Neovim
  - dein
- Zsh
  - zplug
- asdf
  - Ruby
  - Python
  - Go
  - Nodejs
  - Erlang
  - Elixir
- GNU GLOBAL
- Ricty(Mac only)

## Install
```
git clone git@github.com:croixxant/dotfiles-2018.git ~/.dwarf
sh ~/.dwarf/bin/dwarf create
```

## Usage
- dwarf link  
  link dotfiles.
- dwarf install  
  install toolset.
- dwarf create  
  link & install.
